# 🗄️ SUPER_DB_EDITOR_ZDAROV

Легкий и мощный веб-интерфейс для управления базами данных (SQLite/PostgreSQL) на Django.  
Сделано для тех, кто ценит скорость, патриотизм и эффективность.

## Фичи
- **Деловой стиль** с фоном "Под присмотром" 🇷
- **Кликер духа**: кликай на флаг, получай заряд бодрости и летающие буквы Z ✨
- **Управление данными**: Просмотр, добавление, удаление записей через контекстное меню (ПКМ).
- **SQL Терминал**: Выполняй пачки запросов сразу (разделитель `;`).
- **Динамический интерфейс**: Автоматическое определение структуры таблиц.

## Установка

1. Клонируй репозиторий:
   git clone https://github.com/ТВОЙ_НИК/super_db_editor_zdarov.git
   cd super_db_editor_zdarov
2. Установи зависимости:
    pip install -r requirements.txt
3. Примени миграции:
    python manage.py migrate
4. Запусти сервер:
    python manage.py runserver
5. Открой ссылку
    [text](http://127.0.0.1:8000)

## Технологии
- Django + DRF
- Vanilla JS (без тяжелых фреймворков)
- SQLite / PostgreSQL
