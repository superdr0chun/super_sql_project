# SUPER_DB_EDITOR_ZDAROV

## Установка

1. Клонируй репозиторий:
   - git clone https://github.com/superdr0chun/super_sql_project
   - cd super_sql_project
2. Установи зависимости:
    - pip install -r requirements.txt
3. Примени миграции:
    - python manage.py migrate
4. Запусти сервер:
    - python manage.py runserver
5. Открой ссылку:
    - [на свой сервер Django](http://127.0.0.1:8000)

## Технологии
- Django + DRF
- Vanilla JS (без тяжелых фреймворков)
- SQLite / PostgreSQL
